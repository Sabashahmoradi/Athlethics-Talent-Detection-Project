Sprint & Track Talent Identification Project
This repository contains the final project for a sprint & track talent identification system.

## Project Description
The system analyzes athlete information and performance data to:

- Identify talent suitability for sprint and track events
- Analyze basic performance metrics
- Support ranking and prediction of race outcomes
